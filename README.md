## iconPicker
组件名称：iconPicker

功能介绍：用于选择 Layui 的图标
1. 可根据配置方式选择 font-class 或 unicode 两种格式
2. 组件直接获取 layui 的内置图标，省去更新同步问题
3. 并支持清除所选图标功能
4. 支持搜索图标（fontClass 模式下）
5. 支持图标分页和每页数量配置
6. 支持图标渲染成功和点击后的函数回调
